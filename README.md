# Vita3K快速指南
支持的操作系统：
- MacOS
- Ubuntu
- Windows 64位系统（Windows系统则需要安装[Visual C++ 2015-2019 Redistributable运行库](https://aka.ms/vs/16/release/vc_redist.x64.exe)）

最低要求： 
- 处理器：支持AVX指令集
- 显卡：支持OpenGL 4.1（某些游戏的图形可能会出现问题）
- 运行内存：4GB（运行内存2GB也可以运行）

推荐要求： 
- 处理器：支持AVX指令集
- 显卡：支持着色器互锁、支持OpenGL 4.3或者以上版本
- 运行内存：8GB或者以上的运行内存
