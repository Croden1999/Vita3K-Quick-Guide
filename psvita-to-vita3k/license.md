# 授权安装
如果应用程序/游戏缺少license授权文件可能导致无法运行应用（指pkg/nonpdrm zip安装后），安装授权即可正常解密运行。如果在PSVita提取授权或安装过nonpdrm游戏，授权文件会在`ux0:nonpdrm/app/[TITLE_ID]/*.rif`或`ux0:app/[TITLE_ID]/sce_sys/package/work.bin`内存卡路径。
1. 在Vita3K菜单栏选择`文件`，选择`安装授权`；
2. 选择`导入work.bin/rif`或`输入rzif密钥`，授权文件或密钥需要与你要启动的游戏ID对应一致；
3. 完成授权安装后，运行游戏即可；
4. 如果仍然出现运行错误，建议检查并按照步骤重新安装授权。
